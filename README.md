# EnigmaPolLucas

Alumnes: Pol Latorre i Lucas Delas

# De que va?
Aquest programa simula una maquina Enigma de 3 rotors. Permet xifrar i desxifrar missatges i canviar la configuracio dels rotors.

# Com fer-ho servir
1. Els fitxers de text han d'estar dins la carpeta "text".
2. Obre el fitxer "main.py" i executa'l donant-li al Play.
3. El programa et demanara la posicio inicial dels rotors.

# Fitxers
- Entrada: "Missatge.txt" (el text a xifrar) i els fitxers dels rotors.
- Sortida: El resultat es guardara automaticament a "Xifrat.txt" o "desxifrat.txt".
